
# **Simple ToDo List with HTML, CSS, and JavaScript**

**try it:** https://youssef-erremili.github.io/To-Do-list/ 

**Description:**
Introducing our straightforward ToDo list application, built with HTML, CSS, and JavaScript. This ToDo list helps you organize tasks efficiently while incorporating CRUD (Create, Read, Update, Delete) functionality for seamless task management.

**Key Features:**
1. **Minimalist Design:** Our ToDo list boasts a minimalist design, focusing on functionality without unnecessary clutter.
2. **CRUD Operations:** With CRUD functionality, you can easily Create, Read, Update, and Delete tasks as needed, making task management a breeze.
3. **Responsive Layout:** The ToDo list is responsive, ensuring a consistent experience across various devices, whether you're using a desktop, tablet, or smartphone.



**Technologies Used:**
- **HTML:** Provides the structure for the ToDo list, including task items and input fields.
- **CSS:** Styles the ToDo list for a clean and visually appealing interface, enhancing user experience.
- **JavaScript:** Implements CRUD operations and interactivity, allowing users to add, edit, delete, and mark tasks as complete.

**How to Use:**
To use the ToDo list, simply visit the webpage and start adding tasks using the input field. You can mark tasks as complete by checking them off, edit them by clicking on them, or delete them altogether.
